主机管理 批量纳管 Linux 主机（SSH 密钥）

SSH 自检（Windows） 先验证免密登录，再添加到平台

需要准备：目标主机 IP、端口、用户名、本机 OpenSSH 私钥文件路径（例如 C:\Users\你\.ssh\id_ed25519）。
1) 下载脚本：ssh_check.ps1
2) PowerShell 运行示例：
powershell -ExecutionPolicy Bypass -File .\ssh_check.ps1 123.207.215.86 ubuntu C:\Users\你\.ssh\octopus_ed25519 22
3) 若脚本输出的公钥能成功登录，再把 私钥全文 粘贴到下方“SSH 私钥”框中添加主机。
